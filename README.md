![Logo](res/logo.png)

Tool for analyze the image quality

## Requirements 
```
python-opencv
```

### Linux

#### Debian/Ubuntu
```
$ sudo apt-get install python-opencv
$ sudo pip install opencv-python
```

## Usage
Format supported: RGB, YUV

```
$ ./colorscope.py -i image.jpeg -f=RGB
```
